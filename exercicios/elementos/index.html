Elementos Inline

Texto e Formatação

  • <br>
    Destaca o texto em negrito, mas sem semântica especial.

  • <big>
    Aumenta o tamanho da fonte do texto.

  • <i>
    Aplica itálico ao texto sem adicionar semÂntica.

  • <small>
    Reduz o tamanho da fonte do texto.

Elementos Semânticos e de Estilização

  • <abbr>
    Define uma abreviação ou acrônimo, com a possibiliadde de fornecer o significado completo usando o atributo title.

    • HTML

  • <cite>
    Usado para marcar o título de uma obra citada ou uma referência.

  • <code>
    Representa um trecho de código de computador.

  • <dfn>
    Define o termo que está sendo definido na frase.

  • <em>
    Aplica ênfase ao texto, normalmente em itálico, com significado semântico.

  • <kbd>
    Representa uma entrada de teclado.

  • <strong>
    Indica imporância ou enfâse forte ao texto, normalmente rendenizando em negrito.

  • <samp>
    Representa uma saída de computador ou codigo.

  • <var>
    Define uma variável em expessões matemáticas ou código.

Elementos de Interação

  • <a>
    Cria um link de hipertexto.

  • <bdo>
    Controla a direção do texto da esquerda para a direita ou vice-versa.

    • Como por exemplo esse texto está escrito da direita para a esquerda.

  • <br>
    Insere uma quebra
    de linha.

  • <img>
    Insere uma imagem.

  • <map>
    Define um mapa de imagem, usado para criar áreas clicáveis em uma imagem.

  • <object>
    Embeda objetos externos como plugins ou outros documentos.

  • <q>
    Representa uma citação curta (inline)

  • <script>
    Insere ou referência um código de script.

  • <span>
    Usado para agrupar texto ou elementos para estilização (não tem semântica própria).

  • <sub>
    Aplica texto como subscrito.

  • <sup>
    Aplica texto como sobrescrito.

Elementos de Formulário

  • <button>
    Cria um botão interativo.

  • <input>
    Insere divesos tipos de campo de entrada.

  • <label>
    Associa uma descrição textual a um elemento de formulário.

  • <select>
    Cria um menu suspenso.

  • <textarea>
    Cria uma área de texto para entrada de dados multi-linha.
